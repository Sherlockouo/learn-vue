## 组件化

![组件化](https://cdn.jsdelivr.net/gh/Sherlockouo/PicBase/img/learn/组件化.png)

### 1.创建组件构造器

```js
  //1。创建组件构造器对象
  const cmp = Vue.extend({
    template: `
      <div>
         <h1>组件化</h1>
         <h2>内容：乌拉</h2>
      </div>`
  })
```

> ``:可以换行写
>
> ''和“”:换行之后需要加号连接

### 2.注册组件

```js
 //2.注册组件
  Vue.component('my-cpn',cmp)
```



### 3.使用组件

```js
<div id="app">
  //3。使用组件
  <my-cpn></my-cpn>
</div>
```
